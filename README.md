## ✨ 核心功能

| 工具 | 说明 | 适用场景 |
|------|------|---------|
| **BMI 计算器** | 身体质量指数计算，快速判断体重状况 | 日常健康监测 |
| **每日卡路里计算器** | 根据年龄/性别/活动水平计算每日热量需求 | 减肥/增肌 |
| **基础代谢率 (BMR)** | 计算静息状态下消耗的热量 | 科学饮食规划 |
| **体脂率估算器** | 基于身体测量数据估算体脂率 | 精准身体评估 |
| **理想体重计算器** | 综合多种医学公式计算健康体重范围 | 目标设定 |
